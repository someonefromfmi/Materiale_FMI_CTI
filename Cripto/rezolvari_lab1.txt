Ex 6:
Malware
Orice software malitios, creat cu scopul de a dauna unui sistem informatic, a fura date sau a compromite securitatea acestuia.

Virus
Malware care se ataseaza unui fisier legitim si se reproduce automat cand fisierul este executat. Poate deterioara date, incetini sistemul sau permite accesul neautorizat.

Dropper
Malware care contine si instaleaza alte componente malware pe sistemul tinta.

Downloader
Descarca componente malitioase de pe Internet si le instaleaza.

Trojan
Program care pare legitim sau util, dar care contine cod malitios ascuns. Ofera unui posibil atacator acces la sistem sau efectueaza actiuni neautorizate fara stirea utilizatorului.

Spyware
Program are spioneaza activitatea utilizatorului, colectand informatii precum parole, obiceiuri de navigare sau date personale, pe care le trimite la un tert.

Riskware
Aplicatii legitime care, in anumite conditii, pot fi folosite abuziv de atacatori pentru scopuri malitioase.

Ransomware
Malware care cripteaza fisierele victimei si cere o rascumparare pentru a le debloca.

Adware
Software care afiseaza reclame nedorite pe sistemul utilizatorului.

Worm
Malware care se reproduce singur si se raspandeste automat prin retele, fara a avea nevoie de un fisier gazda. Poate consuma resursele retelei sau instala alte programe malitioase.

Obfuscare
Tehnica de ascundere a codului sursa sau a intentiei reale a unui program, pentru a ingreuna analiza si detectarea de catre antivirusuri sau cercetatori in securitate.
